The Bird class represents a bird in my Ruby bird making and bird watching game. The number_of_birds is automatically updated when a new bird is created using new, at which point the initialize method is called and the number of birds incremented. wild_birds is an Array belonging to the entire Bird class that gets populated with birds as they are created. flight_birds is a Hash that contains bird species as keys and Boolean values as values in order to indicate whether or not a given species of bird can fly. For example, the key "penguin" would return a value of false from the Hash whereas a key of "eagle" would return true. Each bird has a name, species that may or may not be able to fly, gender (male if value is true, female if value is false), description, and habitat.
